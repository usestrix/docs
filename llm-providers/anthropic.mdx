## Setup

```bash
export STRIX_LLM="anthropic/claude-sonnet-4-5"
export LLM_API_KEY="sk-ant-..."
```

## Available Models

| Model | Description |
|-------|-------------|
| `anthropic/claude-sonnet-4-5` | Latest Sonnet (recommended) |
| `anthropic/claude-opus-4` | Most capable Claude model |
| `anthropic/claude-3-5-sonnet-20241022` | Previous Sonnet |
| `anthropic/claude-3-opus-20240229` | Previous Opus |

## Get API Key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Navigate to API Keys
3. Create a new key
